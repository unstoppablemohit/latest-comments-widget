Latest Comments Widget Plugin Manual

Installation:

  1. Download the Plugin: Download the Latest Comments Widget plugin from the source where you obtained it. Usually, it will be a zip file containing the plugin files.
  2. Log into WordPress: Log in to your WordPress admin dashboard.
  3. Upload the Plugin:
    • Navigate to Plugins > Add New in the WordPress admin menu.
    • Click on the "Upload Plugin" button.
    • Choose the plugin zip file you downloaded and click "Install Now."
    • After installation, click "Activate Plugin" to activate the Latest Comments Widget plugin.

Configuration:

  1. Add the Latest Comments Widget to Your Sidebar:
    • Once the plugin is activated, go to Appearance > Widgets in the WordPress admin menu.
    • Find the "Latest Comments Widget" in the available widgets.
    • Drag and drop the widget to your desired sidebar area.
  2. Configure the Widget:
    • You will see options to set a title for the widget (optional) and the number of comments to display.
    • Enter a title (if you want to display one) and specify the number of comments you want to show in the widget.
    • Click "Save" to save your widget settings.
  3. View Your Website: Visit your WordPress site to see the Latest Comments Widget in action. The widget will display the latest comments based on your configuration.

Additional Notes:

  • Responsive Design: The widget is designed to be responsive, ensuring it looks good and functions well on various screen sizes, including mobile devices and tablets.
  • Customization (Advanced): If you have CSS knowledge, you can further customize the widget's appearance by modifying the CSS styles within the plugin file. However, it's recommended to create a child theme or use a custom CSS plugin to avoid losing your changes during plugin updates.
  • Support and Documentation: For additional support or questions, refer to the plugin documentation or contact the plugin developer for assistance.

Note: This manual provides basic instructions for installing and configuring the Latest Comments Widget plugin. For the most accurate and detailed information, always refer to the plugin documentation provided by the developer.
